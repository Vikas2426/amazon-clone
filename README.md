## Description

This is personal project. A clone of Amazon's e-commerce website. I have added a few animations too.
You can sign up with your Google account.
Add items to cart. Remove items from cart etc.

#### Technologies

- React
- Context API
- Firebase
- Material UI
- React Router
- React Spring
- CSS
